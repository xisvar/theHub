Module: nouns

Collection of nouns categorized by theme.
Used by the username generator to create the subject part of usernames.
Categories include basic nouns, mythological creatures, animals, nature elements,
and objects/artifacts.

// Sample usage
import { nouns } from './nameBanks.utils.js';
const randomNoun = nouns[Math.floor(Math.random() * nouns.length)];
